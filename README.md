

## โครงสร้างไฟล์

```
namhong-website/
├── index.html                 หน้าแรก
├── about.html                 เกี่ยวกับเรา (ประวัติ/ไทม์ไลน์/วิสัยทัศน์/ผู้บริหาร/โครงสร้างองค์กร/ธรรมาภิบาล)
├── business.html              ภาพรวมธุรกิจ (หน้ากลาง เชื่อม 2 บริษัท)
├── business-palm-oil.html     นามหงส์ น้ำมันปาล์ม
├── business-power.html        นามหงส์ พาวเวอร์
├── sustainability.html        ความยั่งยืน (ESG + วงจรเศรษฐกิจหมุนเวียน)
├── news.html                  ข่าวสาร (มีตัวกรองหมวดหมู่)
├── career.html                ร่วมงานกับเรา (สวัสดิการ/ตำแหน่งงาน/ฟอร์มสมัคร)
├── contact.html                ติดต่อเรา (ฟอร์ม/แผนที่/ที่ตั้ง)
├── privacy-policy.html        นโยบายความเป็นส่วนตัว & PDPA
├── sitemap.xml / robots.txt
├── DEPLOY-TO-HOSTATOM.md      วิธีติดตั้งบน Hostatom ครบทุกขั้นตอน ตั้งแต่สมัครจนเว็บขึ้นจริง (เริ่มอ่านที่นี่)
├── BACKEND-SETUP.md           เอกสารอ้างอิงทางเทคนิคของระบบหลังบ้าน (มีอะไรป้องกันไว้บ้าง/แก้ปัญหาเบื้องต้น)
├── backend/                   ระบบหลังบ้านจริง: รับฟอร์ม → เก็บ MySQL → แดชบอร์ดผู้ดูแลระบบ
└── assets/
    ├── css/style.css          ระบบดีไซน์ทั้งหมด (tokens + components)
    └── js/main.js             ระบบ interactivity ทั้งหมด
```
